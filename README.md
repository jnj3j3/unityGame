# River Racer
## 소개
> River Racer는 강위에서 카트를 운전하여 모든 체크 포인트 링을 지나는 게임입니다.
> 레일 위에는 여러 장애물들이 존재하며 몇 장애물들은 제한 시간을 줄어들게 만듭니다.
